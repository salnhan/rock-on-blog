Easily clear floats by adding .clearfix to the parent element. Can also be used as a mixin.

{% highlight html %}

…

{% endhighlight %}

{% highlight scss %}
// Mixin itself
@mixin clearfix() {
&::after {
display: block;
content: “”;
clear: both;
}
}

// Usage as a mixin
.element {
@include clearfix;
}
{% endhighlight %}
